# Part 15: ලස්සන UI එකක් හදන්නේ කොහොමද? (Tailwind CSS)

අපි දැන් දන්නවා React Component එකක් හදන්න සහ ඒකට මතකයක් දෙන්න. හැබැයි ඒ හදන දේවල් වලට කිසිම හැඩයක් (පාටක්, ප්‍රමාණයක්) නැහැ. වෙබ්සයිට් එකක් ලස්සන කරන්නේ **CSS** කියන තාක්ෂණයෙන්.

සාමාන්‍යයෙන් ඉස්සර කෝඩ් කරද්දී CSS ලිව්වේ වෙනම ෆයිල් එකක (උදා: `style.css`). ඒකෙදි අපි Button එකට `class="btn-primary"` කියලා නමක් දීලා, අර CSS ෆයිල් එකට ගිහින් ඒ නමට අදාළව පාට, දිග, පළල වෙන වෙනම ලිව්වා.

## Tailwind CSS කියන්නේ මොකක්ද?

අපේ UWU-NEXUS Project එකේ පෙනුම (UI) හදන්න අපි පාවිච්චි කරන්නේ සාමාන්‍ය CSS නෙවෙයි, **Tailwind CSS** කියන නවීන ක්‍රමයයි. 

Tailwind වලදී අපිට වෙනම CSS ෆයිල් හදන්න ඕනේ නැහැ. Component එකේ HTML (JSX) කෝඩ් එක ලියන තැනම කෙලින්ම අපිට ඕනේ ස්ටයිල් (Style) එකේ කෙටි නම ලියන්න පුළුවන්. මේකට කියන්නේ **Utility-First CSS** කියලා.

### සරල උදාහරණයක් බලමු:

හිතන්න අපිට නිල් පාට පසුබිමක් (Background) තියෙන, සුදු පාට අකුරු තියෙන, රවුම් වුණු (Rounded) බොත්තමක් හදන්න ඕනේ කියලා.

**1. පරණ සාමාන්‍ය CSS ක්‍රමය:**
```html
<!-- HTML ෆයිල් එකේ -->
<button class="my-btn">Click Me</button>

<!-- CSS ෆයිල් එකේ -->
.my-btn {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border-radius: 8px;
}
```

**2. නවීන Tailwind CSS ක්‍රමය (අපේ Project එකේ ක්‍රමය):**
```tsx
export default function MyButton() {
    return (
        <button className="bg-blue-600 text-white px-5 py-2 rounded-lg">
            Click Me
        </button>
    );
}
```

දැක්කා නේද? අපිට වෙනම CSS ෆයිල් එකකට යන්න වුණේ නැහැ. 
* `bg-blue-600` = නිල් පාට පසුබිම
* `text-white` = සුදු පාට අකුරු
* `px-5 py-2` = ඇතුළත ඉඩ (Padding)
* `rounded-lg` = රවුම් වුණු කොන්

අපේ Project එකේ `app/login/page.tsx` වගේ ඕනෙම ෆයිල් එකකට ගියොත්, ඔයාට මේ වගේ `className="..."` ඇතුළේ ලියපු වචන ගොඩක් දකින්න පුළුවන්. ඒ ඔක්කොම Tailwind CSS කෝඩ්!

## විශේෂ සටහනක් (Inline CSS)

අපේ UWU-NEXUS Project එකේ ගොඩක් තැන් වල Tailwind පාවිච්චි කළත්, සමහර විශේෂ තැන් වල (උදාහරණයක් විදිහට `app/learn/` ෆෝල්ඩරේ තියෙන LMS එකේදී) Tailwind කෝඩ් එක සමහර වෙලාවට නිවැරදිව පේන්නේ නැති ප්‍රශ්නයක් ආවා. 

අන්න ඒ වගේ වෙලාවට අපි කෙලින්ම අදාළ පේළියටම ලියන **Inline CSS (`style={{...}}`)** පාවිච්චි කරලා තියෙනවා. 

**උදාහරණය (Inline CSS):**
```tsx
<p style={{ color: "#374151", fontSize: 18 }}>මෙය සාමාන්‍ය වාක්‍යයකි.</p>
```
ඒ නිසා ඔයා කෝඩ් එක කියවද්දී `className` දැක්කොත් ඒ Tailwind CSS, `style` දැක්කොත් ඒක කෙලින්ම ලියපු සාමාන්‍ය (Inline) CSS කියලා තේරුම් ගන්න.

> **සාරාංශය:**
> * වෙබ්සයිට් එක ලස්සන කරන්නේ CSS වලින්.
> * අපි පාවිච්චි කරන්නේ ලේසියෙන් Component එක ඇතුළෙම ලියන්න පුළුවන් **Tailwind CSS** ය.
> * `bg-red-500`, `text-xl`, `flex` වගේ කෙටි වචන පාවිච්චි කරලා ඉක්මනින් ලස්සන UI එකක් හදාගන්න මේකෙන් පුළුවන්.

මෙතැනින් අපේ "මූලික දැනුම (Basics)" සියල්ල අවසන් වෙනවා! ඔයා දැන් කෝඩ් එකක් දැක්කම ඒක තේරුම් ගන්න පුළුවන් මට්ටමේ (Foundation) ඉන්නවා. 

ඊළඟ පාඩමේ (Module 6) ඉඳන් අපි යන්නේ කෙලින්ම **අපේ Project එකේ Features (Login, Events, Tickets) එකින් එක අරගෙන, ඒවයේ ඇත්තම කෝඩ් එක පේළියෙන් පේළිය කියවන්නයි!**
