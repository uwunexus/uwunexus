# Part 04: PHP කියන්නේ මොකක්ද? (PHP Basics)

කලින් පාඩම් වලින් අපි දැක්කා Backend එක (මොළය) විදිහට අපි පාවිච්චි කරන්නේ **PHP** කියලා. මේ පාඩමෙන් අපි බලමු ඇත්තටම PHP කෝඩ් එකක් ලියන්නේ කොහොමද, ඒක වැඩ කරන්නේ කොහොමද කියලා.

Coding ගැන මුකුත් දන්නේ නැති කෙනෙක් වුණත් මේක කියෙව්වම PHP කෝඩ් එකක් දැක්ක ගමන් බය වෙන්නේ නැතුව ඒකේ මොනවද තියෙන්නේ කියලා තේරුම් ගන්න පුළුවන් වෙයි.

---

## PHP කියන්නේ රහසිගත භාෂාවක්!

පළවෙනි දේ තමයි, PHP කෝඩ් එකක් වැඩ කරන්නේ **Server** එක ඇතුළේ විතරයි. ඒ කියන්නේ අපේ වෙබ්සයිට් එකට එන සාමාන්‍ය කෙනෙක්ට කවදාවත් අපේ PHP කෝඩ් එක බලාගන්න බැහැ. 

ඔයා බ්‍රව්සර් එකෙන් ඉල්ලුවම, PHP එක ඇතුළේ වැඩේ කරලා ඒකේ "ප්‍රතිඵලය" විතරයි බ්‍රව්සර් එකට එවන්නේ. 

## 1. PHP කෝඩ් එකක් පටන් ගන්නේ කොහොමද?

ඕනෑම PHP ෆයිල් එකක් (උදා: `login.php`) අරන් බැලුවොත්, ඒකේ කෝඩ් එක පටන් ගන්නේ සහ ඉවර වෙන්නේ මෙන්න මේ අමුතු සලකුණු දෙකෙන්:

```php
<?php
  // ඔයාගේ කෝඩ් එක ලියන්නේ මෙතනයි
?>
```

මේක දැක්ක ගමන් පරිගණකය දන්නවා "ආ.. මේ තියෙන්නේ PHP කෝඩ් එකක්, මම මේක කියවන්න ඕනේ" කියලා. (`//` දාලා තියෙන්නේ අපි විස්තර ලියන "Comments". ඒවා පරිගණකය ගණන් ගන්නේ නැහැ).

## 2. Echo (කෑගහලා කියන්න!)

PHP වල තියෙන සරලම කමාන්ඩ් එක තමයි `echo` කියන්නේ. මේකෙන් කරන්නේ යම්කිසි දෙයක් Print කරන එක, නැත්නම් බ්‍රව්සර් එකට යවන එක. 

```php
<?php
  echo "Hello UWU-NEXUS!";
?>
```
මෙතනදී පරිගණකය `echo` කියන වචනය දැක්කම, ඒක ඉස්සරහා තියෙන වචන ටික (Hello UWU-NEXUS) අරගෙන තිරයේ පෙන්වනවා. (අනිවාර්යයෙන්ම හැම පේළියක්ම ඉවර වෙද්දී `;` (semicolon) එකක් දාන්න ඕනේ. ඒක හරියට වාක්‍යයක් ඉවර වෙද්දී තිතක් (.) තියනවා වගේ).

## 3. Variables (දත්ත දාන පෙට්ටි)

කෝඩ් එකක් ලියද්දී අපිට එක එක විස්තර (Data) මතක තියාගන්න වෙනවා. ඒවට අපි පාවිච්චි කරන්නේ **Variables**. මේවා හරියට හිස් පෙට්ටි වගේ. අපිට ඕනේ දේවල් ඒ පෙට්ටි ඇතුළට දාලා වහලා, පෙට්ටියට නමක් ගහන්න පුළුවන්.

PHP වල පෙට්ටියකට නමක් දාද්දී අනිවාර්යයෙන්ම **`$` (ඩොලර් ලකුණ)** ඉස්සරහින් දාන්න ඕනේ. 

```php
<?php
  // වචන දාන පෙට්ටියක් (String)
  $name = "Nilesh"; 

  // ඉලක්කම් දාන පෙට්ටියක් (Integer)
  $age = 24; 

  // මේ පෙට්ටි වල තියෙන දේවල් එළියට ගමු (Echo කරමු)
  echo "My name is " . $name . " and I am " . $age . " years old.";
?>
```

මෙතනදී `$name` කියන පෙට්ටිය ඇතුළට අපි "Nilesh" කියන වචනය දැම්මා. 
`$age` කියන පෙට්ටියට 24 දැම්මා.
ඊටපස්සේ `echo` කරද්දී ඒ පෙට්ටි වල තියෙන දේවල් එළියට අරන්, අනිත් වචනත් එක්ක එකතු කරන්න (මුට්ටු කරන්න) අපි **`.` (තිත)** පාවිච්චි කළා. 

### තව උදාහරණයක් බලමු (ගණනය කිරීමක්):

```php
<?php
  $price = 500;
  $quantity = 3;

  $total = $price * $quantity;

  echo "Total amount is: Rs." . $total; 
  // ප්‍රතිඵලය: Total amount is: Rs.1500
?>
```

ඉහත කෝඩ් එකේ `$total` කියන අලුත් පෙට්ටියක් හැදුවා, ඒකට දැම්මේ අර කලින් පෙට්ටි දෙකේ (price සහ quantity) අගයන් දෙක ගුණ කරලා (`*`) ආපු උත්තරයයි.

> **සාරාංශය:**
> * PHP ලියන්නේ `<?php` සහ `?>` ඇතුළෙයි.
> * මොනවා හරි දත්තයක් තියාගන්න ඕනේ නම් අපි ඒක `$` ලකුණෙන් පටන් ගන්න නමක් තියෙන Variable (පෙට්ටියක්) ඇතුළට දානවා.
> * ඒ පෙට්ටියේ තියෙන දේ බලන්න ඕනේ නම් අපි `echo` පාවිච්චි කරනවා.

ඊළඟ පාඩමෙන් අපි බලමු කොහොමද මේ PHP මොළය පාවිච්චි කරලා "තීරණ" ගන්නේ කියලා (If/Else කොන්දේසි).
