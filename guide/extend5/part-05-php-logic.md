# Part 05: PHP වලින් තීරණ ගැනීම (Logic & Functions)

පෙර පාඩමෙන් අපි ඉගෙන ගත්තා Variables (`$name` වගේ) ඇතුළේ දත්ත දාන්න. හැබැයි දත්ත තියාගෙන හිටියට මදි, මොළයක් වුණාම ඒ දත්ත පාවිච්චි කරලා "තීරණ" ගන්න ඕනේ. අද අපි බලමු PHP කොහොමද තීරණ ගන්නේ කියලා.

---

## 1. If/Else (කොන්දේසි දැමීම)

මේක හරියට "ගෙදරට එන කෙනාගේ වයස අවුරුදු 18ට වැඩියි නම් ඇතුළට ගන්න, නැත්නම් එළියට දාන්න" කියනවා වගේ වැඩක්. 
අපේ UWU-NEXUS එකේදි, "ලොග් වෙන්න දාපු පාස්වර්ඩ් එක හරි නම් ඇතුළට ගන්න, නැත්නම් Error එකක් දෙන්න" කියලා තීරණය කරන්නේ මෙන්න මේ `if/else` එකෙනුයි.

```php
<?php
  $age = 20;

  // if (කොන්දේසිය) { කොන්දේසිය හරි නම් වෙන දේ }
  if ($age >= 18) {
      echo "ඔබට ඇතුළු විය හැක!";
  } 
  // else { කොන්දේසිය වැරදි නම් වෙන දේ }
  else {
      echo "ඔබට ඇතුළු වීමට වයස මදි!";
  }
?>
```

මෙතනදී `$age` එක 20යි. 20 කියන්නේ 18ට වඩා ලොකු ගාණක් නිසා `if` එක ඇතුළේ තියෙන දේ වැඩ කරනවා ("ඔබට ඇතුළු විය හැක!").

## 2. Loops (එකම දේ නැවත නැවත කිරීම)

ඔයාට "ආයුබෝවන්" කියලා පේළි 100ක් තිරයේ පෙන්නන්න ඕනේ වුණොත්, `echo "ආයුබෝවන්";` කියලා 100 පාරක් ලියන එක මහ කරදරයක්. ඒ වෙනුවට අපිට පුළුවන් පරිගණකයට කියන්න "මේ වැඩේ 100 පාරක් කරපන්" කියලා. මේකට කියන්නේ **Loop** එකක් කියලා.

බහුලවම පාවිච්චි වෙන Loop ජාතියක් තමයි `foreach` කියන්නේ. විශේෂයෙන් Database එකෙන් දත්ත ගොඩක් (උදාහරණයක් විදිහට ඉවෙන්ට් 10ක්) අරන් ඒ ටික එකින් එක පෙන්නන්න මේක පාවිච්චි කරනවා.

```php
<?php
  // මේක හරියට පෙට්ටි කිහිපයක් තියෙන ලොකු පෙට්ටියක් වගේ (Array)
  $events = array("Welcome Party", "Batch Trip", "Exam Starts");

  // ලොකු පෙට්ටියේ තියෙන දේවල් එකින් එක අරන් $event කියන පොඩි පෙට්ටියට දානවා
  foreach ($events as $event) {
      echo "ඉදිරියට එන උත්සවය: " . $event . "<br>";
  }
?>
```
**ප්‍රතිඵලය:**
ඉදිරියට එන උත්සවය: Welcome Party
ඉදිරියට එන උත්සවය: Batch Trip
ඉදිරියට එන උත්සවය: Exam Starts

## 3. Functions (කෝඩ් එකක් අසුරා තැබීම)

එකම කෝඩ් එක (උදාහරණයක් විදිහට කෙනෙක්ගේ උපන් අවුරුද්දෙන් වයස හොයන කෝඩ් එක) එක එක තැන්වල පාවිච්චි වෙනවා නම්, ඒක ආයේ ආයේ ලියන එක කරදරයි. ඒකට අපි කරන්නේ ඒ කෝඩ් එක ලස්සනට පැකට් කරලා (Pack කරලා) නමක් දානවා. ඒ පැකට් එකට කියන්නේ **Function** එකක් කියලා. ඊටපස්සේ ඕනේ වෙලාවට ඒ නම කියලා පැකට් එක ගෙන්නගන්න පුළුවන්.

```php
<?php
  // Function එකක් හදමු
  function calculateAge($birthYear) {
      $currentYear = 2026;
      $age = $currentYear - $birthYear;
      return $age; // උත්තරය ආපසු යවන්න
  }

  // දැන් අපිට ඕනේ තැනක මේක පාවිච්චි කරන්න පුළුවන්!
  $nileshAge = calculateAge(2000);
  echo "නිලේෂ්ගේ වයස: " . $nileshAge; // ප්‍රතිඵලය: නිලේෂ්ගේ වයස: 26

  $kamalAge = calculateAge(1995);
  echo "කමල්ගේ වයස: " . $kamalAge; // ප්‍රතිඵලය: කමල්ගේ වයස: 31
?>
```

> **සාරාංශය:**
> * තීරණ ගන්න අපි **if / else** පාවිච්චි කරනවා.
> * එකම දේ ආයේ ආයේ කරන්න අපි **Loops (`foreach`)** පාවිච්චි කරනවා.
> * කෝඩ් එකක් පැකට් කරලා නමක් දාලා තියාගන්න අපි **Functions** පාවිච්චි කරනවා.

දැන් ඔයා PHP වල මූලිකම දේවල් ඔක්කොම දන්නවා! ඊළඟ පාඩමෙන් අපි යමු අපේ ඊළඟ ආයුධය වෙන **MySQL Database** එක පැත්තට. දත්ත ගබඩා කරන්නේ කොහොමද කියලා බලමු.
