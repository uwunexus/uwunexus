# Part 07: Database එකත් එක්ක කතා කරමු (SQL සහ CRUD)

කලින් පාඩමෙන් අපි ඉගෙන ගත්තා Database එකක් කියන්නේ Excel ෆයිල් එකක් වගේ දෙයක් කියලා. දැන් අපි බලමු කොහොමද මේ Database එකට අලුතින් විස්තරයක් දාන්නේ, දාලා තියෙන දෙයක් කියවන්නේ නැත්නම් මකලා දාන්නේ කියලා.

PHP (අපේ මොළය) Database එකත් එක්ක කතා කරන්නේ විශේෂ භාෂාවකින්. ඒ භාෂාවට කියන්නේ **SQL (Structured Query Language)** කියලා. මේක හරියට සරල ඉංග්‍රීසි භාෂාව වගේ.

Software ලෝකයේදී Database එකක් එක්ක කරන ප්‍රධාන වැඩ 4ක් තියෙනවා. මේ වැඩ 4ට කෙටියෙන් කියන්නේ **CRUD** කියලා.

* **C - Create** (අලුතින් දත්ත ඇතුළත් කිරීම)
* **R - Read** (ඇති දත්ත කියවීම)
* **U - Update** (ඇති දත්ත වෙනස් කිරීම)
* **D - Delete** (දත්ත මැකීම)

දැන් අපි බලමු මේ වැඩ 4 කරගන්න අපි MySQL වලට කමාන්ඩ් (Queries) දෙන්නේ කොහොමද කියලා. අපි උදාහරණයක් විදිහට `users` කියන Table එක ගමු.

---

## 1. Create - දත්ත ඇතුළත් කිරීම (INSERT)

අලුත් ළමයෙක් කැම්පස් එකට ඇවිත් රෙජිස්ටර් වෙනවා කියන්නේ, අපේ `users` කියන Table එකට අලුත් පේළියක් එකතු වෙනවා කියන එකයි. ඒකට පාවිච්චි කරන්නේ `INSERT INTO` කියන කමාන්ඩ් එකයි.

**SQL කමාන්ඩ් එක:**
```sql
INSERT INTO users (first_name, email, password) 
VALUES ('Nilesh', 'nilesh@gmail.com', '123456');
```
* **තේරුම:** `users` වගුවේ තියෙන first_name, email සහ password කියන තීරු වලට අනුපිළිවෙලින් 'Nilesh', 'nilesh@gmail.com' සහ '123456' කියන දත්ත ටික ඇතුළත් කරන්න.

## 2. Read - දත්ත කියවීම (SELECT)

කවුරුහරි ළමයෙක් ලොග් වෙන්න එනකොට අපි Database එකෙන් අහන්න ඕනේ "මේ ඊමේල් එකෙන් කවුරුහරි ඉන්නවද?" කියලා. ඒකට පාවිච්චි කරන්නේ `SELECT` කමාන්ඩ් එකයි. මේක තමයි වැඩියෙන්ම පාවිච්චි වෙන කමාන්ඩ් එක.

**SQL කමාන්ඩ් එක:**
```sql
-- වගුවේ තියෙන ඔක්කොම දත්ත ගන්න නම්
SELECT * FROM users;

-- අපිට ඕනේ නමක් (first_name) විතරක් ගන්න නම්
SELECT first_name FROM users WHERE email = 'nilesh@gmail.com';
```
* **තේරුම:** `users` වගුවෙන් ඔක්කොම දත්ත (`*` ලකුණෙන් කියන්නේ 'ඔක්කොම' යන්නයි) ගෙනත් දෙන්න.
* දෙවෙනි පේළියේ තේරුම: email එක 'nilesh@gmail.com' වන කෙනාගේ නම විතරක් හොයලා දෙන්න (`WHERE` කියන්නේ කොන්දේසියක් දාන්න පාවිච්චි කරන වචනය).

## 3. Update - දත්ත වෙනස් කිරීම (UPDATE)

ළමයෙක්ට තමන්ගේ පාස්වර්ඩ් එක වෙනස් කරගන්න ඕනේ වුණොත්? අලුතින් පේළියක් දාන්නේ නැහැ, තියෙන පේළිය වෙනස් කරනවා. ඒකට පාවිච්චි කරන්නේ `UPDATE`.

**SQL කමාන්ඩ් එක:**
```sql
UPDATE users 
SET password = 'newpassword123' 
WHERE email = 'nilesh@gmail.com';
```
* **තේරුම:** `users` වගුවේ, ඊමේල් එක 'nilesh@gmail.com' වන කෙනාගේ password එක 'newpassword123' කියලා වෙනස් කරන්න. (මෙතනදී `WHERE` එක නොදැම්මොත්, Database එකේ ඉන්න ඔක්කොම ළමයින්ගේ පාස්වර්ඩ් එක මේකට වෙනස් වෙනවා! ඒ නිසා ගොඩක් පරෙස්සම් වෙන්න ඕනේ).

## 4. Delete - දත්ත මැකීම (DELETE)

කවුරුහරි කැම්පස් එකෙන් අස්වෙලා ගියොත්, එයාගේ විස්තර මකලා දාන්න පාවිච්චි කරන්නේ `DELETE` කමාන්ඩ් එකයි.

**SQL කමාන්ඩ් එක:**
```sql
DELETE FROM users 
WHERE email = 'nilesh@gmail.com';
```
* **තේරුම:** `users` වගුවෙන්, ඊමේල් එක 'nilesh@gmail.com' වන කෙනාව සම්පූර්ණයෙන්ම මකා දමන්න. (මෙහිදීත් `WHERE` එක දැම්මේ නැත්නම් මුළු වගුවම මැකෙනවා).

---

> **සාරාංශය:**
> * දත්ත දාන්න: **`INSERT`**
> * දත්ත බලන්න: **`SELECT`**
> * දත්ත වෙනස් කරන්න: **`UPDATE`**
> * දත්ත මකන්න: **`DELETE`**

මේ ටික දැනගත්තම ඔයාට ලෝකේ තියෙන ඕනෙම Relational Database එකක (MySQL, SQL Server, Oracle) වැඩ කරන්න පුළුවන්. 

අපේ ඊළඟ පාඩමෙන් අපි යන්නේ **Module 3: Object-Oriented Programming (OOP) සංකල්ප** වලට. ඒකෙදි අපි කතා කරමු මොකක්ද මේ OOP කියන්නේ සහ ඒක අපේ Project එකට කොහොමද සම්බන්ධ වෙන්නේ කියලා.
