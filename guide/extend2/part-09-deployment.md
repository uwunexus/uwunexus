# 🎓 UWU-NEXUS සරල මාර්ගෝපදේශය - (Extend Guide)
## 9 වෙනි කොටස: වෙබ් අඩවිය අන්තර්ජාලයට දාන්නේ කොහොමද? (Deployment)

අපි මෙච්චර වෙලා කතා කළේ අපේ පරිගණකය (Computer) ඇතුළේ වෙබ් අඩවිය වැඩ කරන හැටිනේ. හැබැයි ඒක ලෝකෙම ඉන්න අයට බලන්න නම් අන්තර්ජාලයට (Internet) එකට දාන්න ඕනේ. ඒකට කියන්නේ **Deployment** කියලා. මේක කරන්නේ කොහොමද කියලා අපි සරලව බලමු.

---

### 1. Servers (සේවාදායකයන්) කියන්නේ මොනවද?

වෙබ් අඩවියක් අන්තර්ජාලයට දාන්න අපිට අන්තර්ජාලයට සම්බන්ධ කරපු, පැය 24ම වැඩ කරන පරිගණක (Computers) ඕනේ. ඒවට කියන්නේ Servers කියලා. අපේ වෙබ් අඩවිය අන්තර්ජාලයට දාන්න අපි ප්‍රධාන Servers 3ක් පාවිච්චි කරනවා:

1. **GitHub (ගිට්හබ්)**: මේක හරියට අන්තර්ජාලයේ තියෙන පෙන් ඩ්‍රයිව් (Pen Drive) එකක් වගේ. අපේ කෝඩ් එක (Code) සේව් කරලා තියෙන්නේ මෙතන.
2. **Vercel (වර්සෙල්)**: අපේ Frontend එක (Next.js - පේන කොටස) දාලා තියෙන Server එක.
3. **VPS (Virtual Private Server)**: මේක අපිටම වෙන් වුණ Private පරිගණකයක්. අපේ Backend එක (PHP) සහ Database එක තියෙන්නේ මෙතන. (මේකේ IP Address එක තමයි `174.138.28.209`).

### 2. අලුත් දෙයක් එකතු කළාම Auto-Update වෙන්නේ කොහොමද?

හිතන්න ඔයා Frontend එකේ පාටක් වෙනස් කරලා අලුත් වෙබ් අඩවිය අන්තර්ජාලයට දාන්න ඕනේ කියලා. ඔයාට සර්වර් වලට ගිහින් එකින් එක අප්ඩේට් කරන්න ඕනෙ නෑ! මේකට **Auto-Deployment** කියලා ක්‍රමයක් හදලා තියෙනවා.

**වෙන දේ:**
1. ඔයා ඔයාගේ පරිගණකයේ කෝඩ් එක වෙනස් කරලා ඒක **GitHub** එකට යවනවා (Push කරනවා).
2. GitHub එක ඒ වෙලාවෙම **Vercel** එකට කෝල් එකක් දීලා කියනවා, *"මෙන්න අලුත් කෝඩ් එකක් ආවා!"* කියලා.
3. Vercel එක තනියම ඒ අලුත් කෝඩ් එක අරගෙන, වෙබ් අඩවිය අලුත් කරලා විනාඩියක් ඇතුළත අන්තර්ජාලයට දානවා. ඒක හරියට මැජික් එකක් වගේ!

---

### 3. රහස්‍ය දේවල් (Passwords) පරිස්සම් කරන්නේ කොහොමද?

අපේ වෙබ් අඩවියේ Stripe (සල්ලි ගන්න එක) පාස්වර්ඩ් වගේ ගොඩක් රහස්‍ය දේවල් තියෙනවා. ඒවා කෝඩ් එක ඇතුළේ ලියලා GitHub එකට දැම්මොත් වෙන අයට ඒක පේන්න පුළුවන්. ඒක නිසා අපි පාවිච්චි කරන්නේ **Environment Variables (පරිසර විචල්‍ය)**.

මේවා හරියට **"රහස් පෙට්ටි"** වගේ. 
- අපි කෝඩ් එක ඇතුළේ පාස්වර්ඩ් එක ලියන්නේ නෑ. අපි ලියන්නේ *"අර රහස් පෙට්ටියේ තියෙන පාස්වර්ඩ් එක අරගන්න"* කියලා.
- ඒ රහස් පෙට්ටිය (Environment Variables) අපි හදලා තියෙන්නේ Vercel සර්වර් එකේ සහ අපේ VPS සර්වර් එකේ විතරයි. 
- ඒක නිසා කවුරුහරි කෝඩ් එක බැලුවට එයාට පාස්වර්ඩ් එක පේන්නේ නෑ!

### 4. uwunexus.tech කියන නම හැදුනේ කොහොමද?

අන්තර්ජාලයේ තියෙන හැම පරිගණකයකටම (Server) තියෙන්නේ ඉලක්කම් වලින් හදපු නමක්. ඒකට කියන්නේ **IP Address** එක (උදාහරණ: 174.138.28.209). 

හැබැයි ළමයින්ට මේ ඉලක්කම් මතක තියාගන්න අමාරු නිසා අපි සල්ලි දීලා **`uwunexus.tech`** කියන නම (Domain Name එක) අරන් තියෙනවා. ඊට පස්සේ අන්තර්ජාලයේ තියෙන නාමාවලියකට (DNS) ගිහින් අපි කියලා තියෙනවා:
*"කවුරුහරි `uwunexus.tech` කියලා ගැහුවොත් එයාව අපේ Vercel සර්වර් එකට යවන්න"* කියලා.

### 5. ඔයාට වෙනස්කමක් කරන්න ඕනේ නම්?

හිතන්න ඔයාට කවුරුහරි කිව්වොත්:
*"අපිට Stripe (සල්ලි ගන්න එකේ) අලුත් පාස්වර්ඩ් එකක් හම්බුණා, ඒක දාන්න ඕනේ!"*

1. මේක කෝඩ් එකේ වෙනස් කරන්න ඕනේ දෙයක් නෙවෙයි.
2. මේක වෙනස් කරන්න ඕනේ අර "රහස් පෙට්ටියේ". 
3. ඔයා Vercel සර්වර් එකේ Website එකට ලොග් වෙලා, ඒකේ Settings වල තියෙන **Environment Variables** කියන තැනට යන්න ඕනේ.
4. එතන තියෙන පරණ පාස්වර්ඩ් එක මකලා අලුත් එක දාලා Save කරන්න.
