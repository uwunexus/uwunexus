# 🎓 UWU-NEXUS සරල මාර්ගෝපදේශය - (Extend Guide)
## 6 වෙනි කොටස: Marketplace සහ Lost & Found (බඩු විකිණීම සහ ෆොටෝස් දැමීම)

විශ්වවිද්‍යාලයේ ළමයින්ට තමන්ගේ පරණ පොත්, කැල්කියුලේටර් විකුණන්න පුළුවන් තැනක් තමයි **Marketplace (වෙළඳපොළ)**. ඒ වගේම කාගේ හරි දෙයක් නැතිවුණොත් ඒ ගැන දාන්න පුළුවන් තැන තමයි **Lost & Found**. මේවා වැඩ කරන්නේ කොහොමද කියලා අපි සරලව බලමු.

---

### 1. ලොකුම ප්‍රශ්නය: ෆොටෝස් සේව් කරන්නේ කොහොමද?

ළමයෙක් පොතක් විකුණන්න දානකොට ඒකේ ෆොටෝ එකක් දානවනේ. අපි අපේ Database එක (මතක තියාගන්න පොත) ඇතුළේ ෆොටෝස් සේව් කරන්නේ නෑ! 
හේතුව: ෆොටෝස් වලින් ලොකු ඉඩක් යන නිසා අපේ Database එක ඉක්මනට පිරිලා වෙබ් අඩවිය Slow වෙන්න පුළුවන්.

**විසඳුම: Cloudinary (ක්ලවුඩිනරි)**
අපි ෆොටෝස් සේව් කරන්න වෙනමම "Cloud Storage" එකක් (අන්තර්ජාලයේ තියෙන වෙනමම හාඩ් ඩිස්ක් එකක්) පාවිච්චි කරනවා. ඒකේ නම Cloudinary.

**වෙන දේ:**
1. ළමයෙක් ෆොටෝ එකක් අප්ලෝඩ් කරපු ගමන් අපේ වෙබ් අඩවිය ඒක **Cloudinary** එකට යවනවා.
2. Cloudinary එක ඒ ෆොටෝ එක තියාගෙන අපිට ඒ ෆොටෝ එකට යන **ලින්ක් එකක් (URL එකක්)** දෙනවා. (උදාහරණ: `https://cloudinary.com/my-photo.jpg`).
3. ඊට පස්සේ අපේ Backend එක අර ෆොටෝ එක වෙනුවට, **ඒ ලින්ක් එක විතරක්** Database එකේ සේව් කරගන්නවා.
4. කාටහරි ඒක බලන්න ඕනේ වුණාම, Frontend එක අර ලින්ක් එකෙන් ගිහින් ෆොටෝ එක පෙන්නනවා.

---

### 2. Marketplace එක වැඩ කරන්නේ කොහොමද?

මේක හරියටම අපේ Events වගේමයි වැඩ කරන්නේ. 
1. ළමයෙක් අලුත් බඩුවක් විකුණන්න දැම්මම ඒක **"Pending"** විදිහට Database එකේ සේව් වෙනවා.
2. Admin කෙනෙක් ඒක බලලා **"Active"** කළාට පස්සේ තමයි අනිත් අයට ඒක පේන්නේ. 
3. බඩුව විකිණුනාට පස්සේ ළමයාට පුළුවන් ඒක **"Sold" (විකිණී අවසන්)** කියලා මාරු කරන්න. එතකොට ඒක වෙබ් අඩවියෙන් අයින් වෙලා යනවා.

*(මේ බඩු සල්ලි වලට ගන්නේ අපේ වෙබ් අඩවිය හරහා නෙවෙයි. ඒ අයට කතා කරන්න පුළුවන් Phone Number එකක් එතන තියෙනවා, ඒකෙන් කතා කරලා තමා සල්ලි ගෙවන්නේ).*

---

### 3. Lost & Found එකේ වෙනස මොකක්ද?

කෙනෙක්ගේ දෙයක් නැතිවුණාම, ඒක Admin අනුමත කරනකම් ඉන්න ගියොත් පරක්කු වෙන්න පුළුවන්නේ. ඒක නිසා මේකේදී අනුමත කිරීමේ ක්‍රමයක් නෑ.

1. ළමයෙක් "මගේ ෆෝන් එක නැතිවුණා" කියලා දැම්මොත් ඒක කෙලින්ම **"Active"** විදිහට හැමෝටම පේන්න වැටෙනවා.
2. ෆෝන් එක හම්බුණාට පස්සේ එයාට ඒක **"Resolved" (විසඳුනා)** කියලා මාරු කරන්න පුළුවන්.

---

### 4. ඔයාට වෙනස්කමක් කරන්න ඕනේ නම්?

හිතන්න ඔයාට කවුරුහරි කිව්වොත්:
*"ළමයෙක් Marketplace එකට විකුණන්න දාපු දෙයක් වරදදීම 'Sold' කියලා මාක් කරගෙන, ඒක ආපහු පේන්න හදලා දෙන්න!"*

1. Database එකට ගිහින් **`marketplace_items`** කියන Table එක අරින්න.
2. අදාළ ළමයා දාපු බඩුව තියෙන පේළිය හොයාගන්න.
3. ඒකේ `status` කියන තීරුවේ තියෙන "sold" කියන වචනේ මකලා **"active"** කියලා Type කරලා save කරන්න.
4. එච්චරයි! දැන් ඒක ආයෙත් Marketplace එකේ පේන්න පටන් ගන්නවා.

---
**දැන් ඔයාට පින්තූර කොහොමද වෙබ් අඩවියකට දාන්නේ (Cloud Storage) කියලත්, මේ පේජස් දෙකේ වෙනසත් තේරෙනවා ඇති.**

ඊළඟ (7 වෙනි) කොටසෙන් අපි කතා කරමු අපේ වෙබ් අඩවියේ තියෙන ලොකුම මැජික් එකක් වෙන **GPA Calculator (ප්‍රතිඵල ගණනය කරන එක)** කොහොමද වැඩ කරන්නේ කියලා!
