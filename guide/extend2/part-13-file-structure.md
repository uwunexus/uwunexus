# 🎓 UWU-NEXUS සරල මාර්ගෝපදේශය - (Extend Guide)
## 13 වෙනි කොටස: File Structure එක (ෆෝල්ඩර් සහ ෆයිල්ස් තියෙන විදිහ)

අපි මේ වෙබ් අඩවියේ ෆෝල්ඩරයක් ඇරියම ෆයිල්ස් ගොඩක් පේනවා නේද? කෝඩින් දන්නේ නැති කෙනෙක්ට මේවා දැක්කම බය හිතෙන්න පුළුවන්. හැබැයි මේක හරියට **ඔෆිස් එකක තියෙන ෆයිල් කබඩ් එකක්** වගේ පිළිවෙලට තමයි තියලා තියෙන්නේ.

අපි බලමු ප්‍රධාන ෆයිල්ස් මොනවද සහ ඒවයින් මොකද වෙන්නේ කියලා.

---

### 1. `app/` ෆෝල්ඩරය (Frontend එක - පේන කොටස)

මේක ඇතුළේ තමයි වෙබ් අඩවියේ පේන හැම පිටුවක්ම තියෙන්නේ. මේකේ තියෙන ෆයිල්ස් ඔක්කොම වගේ `.tsx` කියන ජාතියේ ඒවා.

- **`app/page.tsx`**: මේක තමයි **මුල් පිටුව (Home Page)**. වෙබ් අඩවියට ආපු ගමන් පේන ලොකු පින්තූරය, ලස්සන වචන ටික තියෙන්නේ මෙතන.
- **`app/layout.tsx`**: මේක හරියට **රාමුවක් (Frame)** වගේ. හැම පිටුවකටම උඩින් තියෙන මෙනුව (Navigation Bar) සහ පල්ලෙහා තියෙන කොටස (Footer) සෙට් කරන්නේ මේ ෆයිල් එකෙන්.
- **`app/globals.css`**: වෙබ් අඩවියට ලස්සන පාට දාන්න පාවිච්චි කරන **පාට පෙට්ටිය** තමයි මේක.
- **`app/login/page.tsx`**: ළමයින්ට Login වෙන්න තියෙන පිටුව.
- **`app/events/page.tsx`**: උත්සව (Events) පෙන්නන පිටුව.
- **`app/components/`**: මේක ඇතුළේ තියෙන්නේ අර අපි කලින් කතා කරපු **Lego කෑලි (Components)**. උදාහරණයක් විදිහට, `NavBar.tsx` (උඩ මෙනුව), `EventCard.tsx` (උත්සවයක විස්තර පෙන්නන පොඩි කොටුව).

---

### 2. `backend/` ෆෝල්ඩරය (Backend එක - වැඩ කරන මොළය)

මේක ඇතුළේ තියෙන්නේ දත්ත එක්ක වැඩ කරන ෆයිල්ස්. මේවා ඔක්කොම `.php` කියන ජාතියේ ඒවා.

- **`backend/db.php`**: මේක තමයි **යතුර!** Frontend එකට Database එකට ඇතුල් වෙන්න දෙන යතුර සහ කනෙක්ෂන් එක හදන්නේ මේ ෆයිල් එකෙන්.
- **`backend/login.php`**: ළමයෙක් Email එකයි Password එකයි ගැහුවම ඒක හරිද කියලා Database එකෙන් බලන ෆයිල් එක.
- **`backend/get_events.php`**: Database එකේ තියෙන උත්සව (Events) වල විස්තර ටික අරගෙන Frontend එකට දෙන ෆයිල් එක.
- **`backend/create_ticket.php`**: කවුරුහරි සල්ලි දීලා ටිකට් එකක් ගත්තම ඒක Database එකේ ලියන ෆයිල් එක.

---

### 3. වෙනත් වැදගත් ෆයිල්ස්

- **`.env.local`**: මේක තමයි අර අපි කලින් කතා කරපු **"රහස් පෙට්ටිය"**. පාස්වර්ඩ්, API Keys ඔක්කොම තියෙන්නේ මෙතන. මේක අනිත් අයට පේන්නෙ නෑ.
- **`next.config.ts`**: මේක හරියට **ට්‍රැෆික් පොලිස්කාරයා** වගේ. "මෙතනට එන්න, අතනට යන්න" කියලා Frontend එකට පාර පෙන්නන රූල්ස් (Rules) තියෙන්නේ මෙතන.
- **`package.json`**: මේක හරියට **සාප්පු ලැයිස්තුවක් (Shopping List)** වගේ. අපේ වෙබ් අඩවිය හදන්න අරන් තියෙන පිටින් ගෙනාපු ටූල්ස් (Stripe, Cloudinary වගේ දේවල්) මොනවද කියලා මේකේ ලියලා තියෙනවා.

---

**සරලයි නේද?** 
ඔයාට මොකක් හරි පිටුවක හැඩය වෙනස් කරන්න ඕනේ නම් `app/` එකට යන්න. Database එකෙන් දත්ත ගන්න විදිහ වෙනස් කරන්න ඕනේ නම් `backend/` එකට යන්න. පාස්වර්ඩ් එකක් වෙනස් කරන්න ඕනේ නම් `.env` එකට යන්න.
