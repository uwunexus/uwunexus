# 🎓 UWU-NEXUS සරල මාර්ගෝපදේශය - (Extend Guide)
## 14 වෙනි කොටස: Frontend එකයි Backend එකයි සම්බන්ධ වෙන්නේ කොහොමද?

අපි කිව්වනේ Frontend එක කියන්නේ පේන දේ, Backend එක කියන්නේ පිටිපස්සේ වැඩ කරන දේ කියලා. හැබැයි මේ දෙක තියෙන්නේ තැන් දෙකකනේ. එතකොට මේ දෙගොල්ලෝ එකිනෙකා එක්ක කතා කරන්නේ (Connect වෙන්නේ) කොහොමද?

මේක වෙන්නේ **API (Application Programming Interface)** කියන පාලමක් හරහා.

---

### 1. පාලම හරහා කතා කරන භාෂාව: JSON

ෆ්‍රාන්ස් ජාතිකයෙකුයි, චීන ජාතිකයෙකුයි කතා කරන්න ගියොත් තේරෙන්නේ නෑනේ. එයාලා දෙන්නම ඉංග්‍රීසි වලින් කතා කළොත් තේරෙනවා. 
අන්න ඒ වගේ Frontend එක කතා කරන්නේ වෙනස් භාෂාවක් (JavaScript). Backend එක කතා කරන්නේ තව භාෂාවක් (PHP). 

මේ දෙගොල්ලෝ එකිනෙකා එක්ක දත්ත හුවමාරු කරගන්න පාවිච්චි කරන "පොදු භාෂාවට" කියන්නේ **JSON (ජේසන්)** කියලා. මේක හරියට පොඩි ටෙක්ස්ට් මැසේජ් එකක් වගේ.

---

### 2. කතා කරන්නේ කොහොමද? (උදාහරණයක් බලමු)

හිතන්න ළමයෙක් වෙබ් අඩවියේ **"Events"** පිටුවට ගියා කියලා. එතනදි වෙන දේ පියවරෙන් පියවර බලමු:

**පියවර 1: Frontend එකෙන් කෝල් එකක් ගැනීම (The Request)**
Frontend එක (Next.js) පාලම හරහා Backend එකට කෝල් එකක් දෙනවා. 
මේකට කෝඩින් භාෂාවෙන් කියන්නේ **"Fetch කරනවා"** කියලා. 
*උදාහරණ කෝඩ් එක:* `fetch('https://uwunexus.tech/api/backend/get_events.php')`

**පියවර 2: Backend එකෙන් කෝල් එක ආන්සර් කිරීම (The Processing)**
Backend එකේ තියෙන `get_events.php` කියන ෆයිල් එක මේ කෝල් එක ගන්නවා. 
ඊට පස්සේ එයා Database එකට ගිහින් අහනවා *"මට Events ටික දෙන්න"* කියලා.

**පියවර 3: Backend එකෙන් උත්තරේ යැවීම (The Response)**
Database එකෙන් විස්තර ටික දුන්නට පස්සේ, Backend එක ඒක අර **JSON** කියන පොදු භාෂාවට හරවනවා.
ඒක පේන්නේ මේ වගේ:
```json
[
  { "නම": "අවුරුදු උත්සවය", "දිනය": "2026-04-10" },
  { "නම": "පොත් සල්පිල", "දිනය": "2026-05-01" }
]
```
ඊට පස්සේ මේ පාර්සලේ අර පාලම හරහා ආපහු Frontend එකට යවනවා.

**පියවර 4: Frontend එකෙන් ඒක පෙන්නීම (The Display)**
Frontend එකට මේ පාර්සලේ (JSON එක) හම්බුණාම එයා ඒක ලස්සනට කඩලා අරන්, අපි දකින ලස්සන කොටු (Cards) ඇතුළට දාලා තිරයේ පෙන්නනවා.

---

### 3. Proxy (ප්‍රොක්සි) කියන්නේ මොකක්ද? (මැදිහත්කරුවා)

සමහර වෙලාවට Frontend එක එක සර්වර් එකක (Vercel) සහ Backend එක තව සර්වර් එකක (VPS) තියෙනකොට කෝල් එකක් ගන්න ගියොත් බ්‍රවුසරය බයවෙලා ඒක බ්ලොක් කරනවා. (මේකට කියන්නේ CORS Error කියලා).

ඒක නවත්තන්න අපි **Proxy (මැදිහත්කරුවෙක්)** පාවිච්චි කරනවා.
ඒ කියන්නේ Frontend එක කෙලින්ම අනිත් සර්වර් එකට කෝල් කරන්නේ නෑ. එයා කෝල් කරන්නේ එයාගේම සර්වර් එකට (`/api/backend/...`). ඊට පස්සේ ඒ සර්වර් එක හොරෙන්ම ගිහින් Backend සර්වර් එකෙන් දත්ත ටික අරන් ඇවිත් දෙනවා.

අපේ `next.config.ts` කියන ෆයිල් එකේ තමයි මේ පාර පෙන්නන ට්‍රැෆික් පොලිස්කාරයා (Proxy එක) ඉන්නේ!

---
**දැන් ඔයාට Frontend එක සහ Backend එක වෙන් වෙලා තිබ්බත් ඒ දෙක "JSON" කියන භාෂාවෙන් කතා කරලා එකට වැඩ කරන විදිහ ගැන හොඳ අදහසක් තියෙනවා.**
