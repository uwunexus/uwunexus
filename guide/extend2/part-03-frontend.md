# 🎓 UWU-NEXUS සරල මාර්ගෝපදේශය - (Extend Guide)
## 3 වෙනි කොටස: Frontend එක (ඉස්සරහා පෙනෙන දේ) වෙනස් කරන්නේ කොහොමද?

අපි කලින් කතා කළානේ Frontend එක කියන්නේ "රෙස්ටුරන්ට් එකක කෑම කන තැන" (අපිට ඇහැට පේන ලස්සන දේවල් තියෙන තැන) කියලා. අද අපි බලමු මේවා හැදිලා තියෙන්නේ කොහොමද සහ අපිට ඕනේ නම් ඒකේ තියෙන දෙයක් වෙනස් කරන්නේ කොහොමද කියලා.

---

### 1. Frontend එක හදලා තියෙන්නේ මොකෙන්ද?

මේක හදන්න අපි පාවිච්චි කරලා තියෙන්නේ **Next.js (React)** කියන තාක්ෂණය. 
මේක හරියට **Lego (ලෙගෝ) බ්ලොක්ස් වලින් ගෙයක් හදනවා වගේ** වැඩක්. අපි පොඩි පොඩි කෑලි (Components) හදලා, ඒවා එකතු කරලා තමයි සම්පූර්ණ පිටුවක් හදන්නේ.

උදාහරණයක් විදිහට වෙබ් අඩවියේ උඩින්ම තියෙන මෙනුව (Navigation Bar) එක Lego කෑල්ලක්. ඒ කෑල්ල හැම පිටුවකටම උඩින් අලවලා තියෙනවා.

---

### 2. Frontend ෆයිල්ස් තියෙන්නේ කොහෙද?

project ෆෝල්ඩරය ඇතුළේ තියෙන **`app` ෆෝල්ඩරය** ඇතුළේ තමයි මේ ඔක්කොම තියෙන්නේ.

- මුල් පිටුව (Home Page) = `app/page.tsx`
- Login පිටුව = `app/login/page.tsx`
- Events පිටුව = `app/events/page.tsx`
- Navigation Bar (මෙනුව) = `app/components/NavBar.tsx`

*(මේ `.tsx` කියන්නේ මේවා හැදිලා තියෙන භාෂාවට කියන නමයි.)*

---

### 3. උදාහරණ 1: මුල් පිටුවේ වචනයක් වෙනස් කරමු!

හිතන්න ඔයාට කවුරුහරි කිව්වොත්:
*"අපේ මුල් පිටුවේ (Home Page) ලොකුවට ලියලා තියෙනවා 'Welcome to UWU-NEXUS' කියලා. ඒක 'ආයුබෝවන් UWU-NEXUS' කියලා සිංහලෙන් වෙනස් කරන්න!"*

මේක කරන්නේ මෙහෙමයි:
1. `app` ෆෝල්ඩරයට ගිහින් `page.tsx` කියන ෆයිල් එක Notepad වගේ එකකින් අරින්න. (VS Code වගේ සොෆ්ට්වෙයා එකක් වඩාත් හොඳයි).
2. ඒක ඇතුළේ ලියලා තියෙන කෝඩ් එක අස්සේ ඔය 'Welcome to UWU-NEXUS' කියන වචනේ හොයන්න.
3. ඒක මකලා 'ආයුබෝවන් UWU-NEXUS' කියලා ටයිප් කරන්න.
4. ෆයිල් එක Save කරන්න. එච්චරයි! දැන් වෙබ් අඩවියට ගිහින් බැලුවම ඒක වෙනස් වෙලා තියෙනවා පෙනෙයි.

---

### 4. උදාහරණ 2: පාටක් වෙනස් කරන්නේ කොහොමද?

හිතන්න බොත්තමක පාට නිල් පාටින් තියෙනවා. ඒක රතු පාට කරන්න ඕනේ.
අපේ project එකේ පාට දාන්න පාවිච්චි කරලා තියෙන්නේ **Tailwind CSS** කියන ක්‍රමය. ඒ කියන්නේ කෝඩ් එක ඇතුළෙම පොඩි වචනයක් ලියලා පාට දෙන එක.

ඔයාට කෝඩ් එක ඇතුළේ මෙහෙම එකක් පෙනෙයි:
`<button className="bg-blue-500 text-white"> Click Me </button>`

- `bg-blue-500` කියන්නේ "Background එක නිල් පාටයි" කියන එක.
- ඔයාට ඒක රතු පාට කරන්න ඕනේ නම් `bg-blue-500` මකලා `bg-red-500` කියලා ලියන්න විතරයි තියෙන්නේ!

---

### 5. ලෙගෝ (Lego) කෑලි වල මැජික් එක!

හිතන්න වෙබ් අඩවියේ උඩින්ම තියෙන මෙනුවේ (Navigation Bar) අලුත් button එකක් දාන්න ඕනේ කියලා.
අපේ වෙබ් අඩවියේ පිටු 10ක් විතර තියෙනවා. ඒ පිටු 10ටම ගිහින් ඒක වෙනස් කරන්න ඕනෙද? **නැහැ!**

මතකද මම කිව්වා මේවා Lego කෑලි වගේ කියලා? 
Navigation Bar එක කියන්නේ එක කෑල්ලක්. ඒක තියෙන්නේ `app/components/NavBar.tsx` කියන ෆයිල් එකේ.
ඔයා ඒ ෆයිල් එක එක පාරක් අරගෙන අලුත් button එකක් ලිව්වොත්, ඒක **වෙබ් අඩවියේ තියෙන පිටු 10ටම** එකපාරින් ඇඩ් වෙනවා! ඒක තමයි මේකේ තියෙන ලොකුම වාසිය.

---
**දැන් ඔයා දන්නවා වෙබ් අඩවියක පේන දේවල් වෙනස් කරන්නේ කොහොමද කියලා.**

ඊළඟ (4 වෙනි) කොටසෙන් අපි කතා කරමු **Backend (පිටිපස්සේ වැඩ කරන මොළය)** එකෙන් මේ වචනයි පාටයි නැතුව ඇත්තටම දත්ත අරගෙන වැඩ කරන්නේ කොහොමද කියලා.
