# Part 02: UWU-NEXUS කියන්නේ මොකක්ද? (Project Overview)

පළමු පාඩමෙන් අපි Frontend, Backend සහ Database කියන්නේ මොනවාද කියලා ඉගෙන ගත්තා. දැන් අපි බලමු අපි හදන්න යන, නැත්නම් වෙනස් කරන්න යන මේ **UWU-NEXUS** කියන මෘදුකාංගය (Software එක) ඇත්තටම මොකක්ද කියලා.

## UWU-NEXUS යනු කුමක්ද?

සරලවම කිව්වොත් UWU-NEXUS කියන්නේ **විශ්වවිද්‍යාල සිසුන්ගේ සියලුම අවශ්‍යතා එකම තැනකින් සපුරාගන්න පුළුවන් Digital Platform එකක්**. 

කැම්පස් එකේ ළමයින්ට ඉවෙන්ට් ගැන දැනගන්න, පරණ පොත් විකුණන්න, නැතිවූ දේවල් හොයාගන්න, තමන්ගේ GPA එක ගණනය කරන්න මේ හැමදේටම වෙන වෙනම ඇප් පාවිච්චි කරන්නේ නැතුව එකම තැනකින් මේ ඔක්කොම කරගන්න පුළුවන් තැනක් තමයි මේක.

---

## UWU-NEXUS හි ප්‍රධාන කොටස් (Features)

ඔබ මේ Project එකේ කේතයන් (Code) වෙනස් කරන්න කලින්, මේකෙ තියෙන ප්‍රධාන කොටස් මොනවාද කියලා දැනගෙන ඉන්න එක අනිවාර්යයි. අපේ Project එක ප්‍රධාන කොටස් කිහිපයකට බෙදලා තියෙනවා:

### 1. ගිණුම් කළමනාකරණය (Authentication & Users)
මේක තමයි පදනම. ළමයින්ට තමන්ගේ ඊමේල් එක දීලා ලියාපදිංචි වෙන්න (Sign up), ලොග් වෙන්න (Login) වගේම පාස්වර්ඩ් අමතක වුණාම අලුත් එකක් හදාගන්න (Forgot Password) මෙතනින් පුළුවන්. ඒ වගේම ළමයින්ට සහ Admin (පරිපාලක) කෙනෙක්ට වෙනස් බලතල දෙන එකත් කරන්නේ මෙතනින්.

### 2. Event Management (උත්සව සහ දැනුම්දීම්)
කැම්පස් එකේ තියෙන අලුත් ඉවෙන්ට්, පාටි, බෝඩිං මීටිං වගේ දේවල් දාන්න සහ අනිත් අයට ඒක බලන්න පුළුවන් තැන.

### 3. Ticket Purchasing (ටිකට් මිලදී ගැනීම)
සමහර ඉවෙන්ට් වලට සල්ලි දීලා ටිකට් ගන්න ඕනේ. ඒ වගේ වෙලාවට කාඩ් එකෙන් (Visa/Mastercard) හරි PayHere හරහා හරි සල්ලි ගෙවලා ටිකට් එකක් ගන්න මේ පද්ධතිය හරහා පුළුවන්. මේක ටිකක් සංකීර්ණ, හැබැයි ගොඩක් වැදගත් කොටසක්.

### 4. Marketplace (වෙළඳපොළ)
ළමයින්ට තමන්ට අවශ්‍ය නැති පරණ පොත්, ලැප්ටොප්, බෝඩිං බඩු වගේ දේවල් විකුණන්න දාන්නත්, අනිත් ළමයින්ට ඒවා සල්ලි දීලා ගන්නත් පුළුවන් Campus කඩයක් වගේ තැනක්.

### 5. Lost & Found (නැතිවූ දේවල්)
කාගේ හරි පෑනක්, පොතක්, ID එකක් නැතිවුණොත් "මගේ මේක නැතිවුණා" කියලා දාන්නත්, කෙනෙක්ට මොනවා හරි අහුලන්න ලැබුණොත් "මට මේක හම්බුණා" කියලා දාන්නත් පුළුවන් තැන.

### 6. GPA Calculator (ප්‍රතිඵල ගණනය)
විශ්වවිද්‍යාල සිසුන්ට තමන්ගේ විෂයයන් වල රිසාල්ට් (A, B, C) ඇතුළත් කළාම, තමන්ගේ GPA (Grade Point Average) එක කීයද කියලා ලේසියෙන්ම බලාගන්න පුළුවන් කැල්කියුලේටරයක්.

### 7. Info Hub (දැනුම ගබඩාව)
කැම්පස් එකේ නීතිරීති, විභාග ගැන විස්තර වගේ වැදගත් ලිපි දාන්න තියෙන තැනක්.

### 8. Admin Panel (පාලක මැදිරිය)
සාමාන්‍ය ළමයින්ට කරන්න බැරි දේවල් - උදාහරණයක් විදිහට වැරදි විදිහට දාලා තියෙන පෝස්ට් මකන්න, ළමයින්ව අයින් කරන්න වගේ සුපිරි බලතල තියෙන Admin (පරිපාලක) කෙනෙක්ට විතරක් යන්න පුළුවන් විශේෂ කොටස.

## මේ Features එකට එකතු වෙන්නේ කොහොමද?

මෙන්න මේ හැම Feature එකක්ම පිටිපස්සේ ඊයේ අපි කතා කරපු **"Frontend - Backend - Database"** කියන තුන් කට්ටුව වැඩ කරනවා. 

*උදාහරණයක් විදිහට අපි **Marketplace** එක ගත්තොත්:*
1. **Frontend එකේ:** ඔයාට පේනවා ලස්සන පොතක පින්තූරයක් සහ "Buy Now" කියලා බොත්තමක් (මේක ලියලා තියෙන්නේ Next.js වලින්).
2. **Backend එකේ:** ඔයා Buy කියන එක එබුවම PHP වලින් ලියපු කෝඩ් එකක් ක්‍රියාත්මක වෙලා බලනවා ඔයාට සල්ලි තියෙනවද කියලා.
3. **Database එකේ:** MySQL Database එකේ ගබඩා කරලා තියෙනවා "කවුද පොත විකුණන්නේ", "කීයද ගාණ" වගේ දේවල්. PHP එක Database එකට කතා කරලා තමයි ඒ විස්තර ගන්නේ.

> **සාරාංශය:**
> UWU-NEXUS කියන්නේ පොඩි වෙබ්සයිට් එකක් නෙවෙයි, Features ගොඩක් තියෙන ලොකු පද්ධතියක්. හැබැයි බය වෙන්න එපා! මේ හැම Feature එකක්ම වැඩ කරන්නේ එකම මූලික න්‍යායකටයි. එකක් තේරුණාම අනිත් ඔක්කොම තේරුම් ගන්න ලේසියි.

අපේ ඊළඟ පාඩමෙන් අපි බලමු මේ ලොකු පද්ධතිය හදන්න අපි පාවිච්චි කරපු **Next.js, PHP, සහ MySQL** කියන තාක්ෂණික මෙවලම් (Tech Stack) හරියටම මොනවාද කියලා.
