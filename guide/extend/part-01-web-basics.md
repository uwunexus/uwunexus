# Part 01: Web එකක් වැඩ කරන්නේ කොහොමද? (Web Basics)

කේතනය (Coding) ගැන කිසිවක් නොදන්නා ඔබට, මේ ලෝකයේ වෙබ් අඩවි වැඩ කරන්නේ කොහොමද කියලා තේරුම් ගන්න එක තමයි මුල්ම පියවර. 

මෙය තේරුම් ගන්න අපි සරල උදාහරණයක් ගමු: **ඔබ ආපනශාලාවකට (Restaurant එකකට) යනවා.**

මෙම ආපනශාලාවේ ප්‍රධාන කොටස් 3ක් තියෙනවා:
1. **ඔබ වාඩි වී ඉන්න ලස්සන කෑම කාමරය (Dining Area)**
2. **ඔබේ ඕඩරය අරන් ගිහින් කෑම හදන කෝකියා (The Chef)**
3. **කෑම හදන්න අවශ්‍ය බඩු ගබඩා කරලා තියෙන කාමරය (Store Room)**

පරිගණක ලෝකයේදී, වෙබ් අඩවියක් වැඩ කරන්නෙත් හරියටම මේ විදිහටමයි! ඒකෙත් ප්‍රධාන කොටස් 3ක් තියෙනවා.

---

## 1. Frontend (කෑම කාමරය)

ඔබ බ්‍රව්සරයෙන් (Chrome, Safari වගේ) වෙබ් අඩවියකට ගියාම ඔබේ ඇහැට දකින්න ලැබෙන **ලස්සන පෙනුම, පාට, බොත්තම්, පින්තූර** සියල්ලම අයිති වෙන්නේ **Frontend** එකටයි. 
මේක හරියට ආපනශාලාවේ ඔබ වාඩි වෙලා ඉන්න ලස්සන කෑම කාමරය වගේ. 

* **පාවිච්චි කරන භාෂා:** HTML, CSS, JavaScript 
* **අපේ Project එකේදී (UWU-NEXUS):** අපි Frontend එක හදන්න පාවිච්චි කරන්නේ **Next.js (React)** කියන නවීන තාක්ෂණයයි.

## 2. Backend (කෝකියා)

ඔබ Frontend එකේ තියෙන "Login" බොත්තම එබුවම, ඒ පණිවිඩය අරගෙන ගිහින් හරිද වැරදිද බලලා තීරණ ගන්නේ **Backend** එකයි. 
මේක හරියට ඔබ වේටර්ට දෙන ඕඩරය අරන් ගිහින් උයන කෝකියා වගේ. කෝකියා කරන දේවල් ඔබට පේන්නේ නැහැ, හැබැයි එයා තමයි ප්‍රධාන වැඩේ කරන්නේ.

* **පාවිච්චි කරන භාෂා:** PHP, Python, Node.js, Java
* **අපේ Project එකේදී (UWU-NEXUS):** අපි Backend එක හදන්න පාවිච්චි කරන්නේ **PHP** කියන භාෂාවයි.

## 3. Database (බඩු ගබඩාව)

ඔබේ පාස්වර්ඩ් එක, ඔබේ පින්තූර, ඉවෙන්ට් විස්තර වගේ හැම දෙයක්ම ආරක්ෂිතව තැන්පත් කරලා තියෙන්නේ **Database (දත්ත ගබඩාව)** එකේ. 
මේක හරියට කෝකියාට කෑම උයන්න අවශ්‍ය බඩු ගෙනත් දෙන ගබඩා කාමරය වගේ. කෝකියාට (Backend) අවශ්‍ය වුණාම ගබඩාවෙන් (Database) බඩු (Data) අරන් එනවා.

* **පාවිච්චි කරන භාෂා:** SQL (MySQL, PostgreSQL)
* **අපේ Project එකේදී (UWU-NEXUS):** අපි දත්ත ගබඩා කරන්න පාවිච්චි කරන්නේ **MySQL** ය.

## මේ තුන එකතුවෙලා වැඩ කරන්නේ කොහොමද? (The Flow)

මෙන්න මේ විදිහට තමයි ඔයා ෆේස්බුක් හරි අපේ UWU-NEXUS හරි පාවිච්චි කරද්දී ඇතුළේ වැඩේ වෙන්නේ:

1. **Frontend:** ඔයා ඔයාගේ Email එකයි Password එකයි ගහලා "Login" බොත්තම ඔබනවා.
2. **Backend:** Frontend එකෙන් මේ විස්තර ටික අපේ කෝකියාට (PHP වලට) යවනවා.
3. **Database:** කෝකියා (PHP) ගිහින් ගබඩාවෙන් (MySQL) අහනවා "මෙහෙම ඊමේල් එකක් සහ පාස්වර්ඩ් එකක් තියෙන කෙනෙක් ඉන්නවද?" කියලා.
4. **Backend:** ගබඩාවෙන් (MySQL) කියනවා "ඔව්, එහෙම කෙනෙක් ඉන්නවා" කියලා. කෝකියා (PHP) ඒ පණිවිඩය ආපහු Frontend එකට යවනවා.
5. **Frontend:** ඔයාගේ ඇහැට පේන Screen එක වෙනස් වෙලා ඔයාව ඇතුළට (Dashboard එකට) අරන් යනවා!

> **සාරාංශය:**
> ඔයාට පේන දේ **Frontend (Next.js)**. 
> හිතන, තීරණ ගන්න මොළය **Backend (PHP)**. 
> විස්තර මතක තියාගන්න තැන **Database (MySQL)**.

මේක තේරුණා නම්, ඔයාට Software Development වල ලොකුම පදනම වැටිලා ඉවරයි! ඊළඟ පාඩමෙන් අපි බලමු අපේ UWU-NEXUS Project එක කියන්නේ හරියටම මොකක්ද කියලා.
