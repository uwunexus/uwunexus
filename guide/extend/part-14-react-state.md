# Part 14: React වල මතකය රඳවා ගැනීම (`useState` සහ `useEffect`)

කලින් පාඩමෙන් අපි ඉගෙන ගත්තා React Component එකක් (ලෙගෝ කෑල්ලක්) හදන්නේ කොහොමද කියලා. හැබැයි ඒ හදපු Component එකට කිසිම "මතකයක්" (Memory එකක්) නැහැ. 

උදාහරණයක් විදිහට, අපි "Like" කියලා බොත්තමක් හැදුවොත්, ඒක කී පාරක් එබුවද කියලා Component එකට මතක තියාගන්න බැහැ. අන්න ඒ මතකය ලබා දෙන්න තමයි අපි **React Hooks** කියන විශේෂ මෙවලම් පාවිච්චි කරන්නේ.

අපේ UWU-NEXUS Project එකේ හැමතැනම පාවිච්චි වෙන ප්‍රධානම Hooks දෙකක් තියෙනවා. ඒ තමයි `useState` සහ `useEffect`.

---

## 1. `useState` (මතක තබාගැනීමේ පෙට්ටිය)

`useState` කියන්නේ Component එක ඇතුළේ දත්ත මතක තියාගන්න පාවිච්චි කරන "මැජික් පෙට්ටියක්" වගේ දෙයක්. සාමාන්‍ය Variable එකකට (`let x = 0`) වඩා මේක විශේෂ වෙන්නේ, මේ පෙට්ටියේ තියෙන දේ වෙනස් වුණ ගමන්ම, **බ්‍රව්සර් එකේ තිරය (Screen එක) ඉබේම අලුත් වෙනවා (Auto Refresh වෙනවා)**.

අපි සරල Like බොත්තමක් හදමු:

```tsx
"use client"; // Next.js වල මේක දැම්මම තමයි Hooks පාවිච්චි කරන්න පුළුවන්

import { useState } from "react";

export default function LikeButton() {
    // likes කියන්නේ මතක තියාගන්න පෙට්ටිය.
    // setLikes කියන්නේ ඒ පෙට්ටියට අලුත් අගයක් දාන රිමෝට් කන්ට්‍රෝල් එක.
    // (0) කියන්නේ පෙට්ටියේ මුලින්ම තියෙන අගය.
    const [likes, setLikes] = useState(0);

    const handlePress = () => {
        setLikes(likes + 1); // බොත්තම එබුවම තියෙන ගාණට 1ක් එකතු කරනවා
    };

    return (
        <div>
            <p>Likes ගණන: {likes}</p>
            <button onClick={handlePress}>Like කරන්න!</button>
        </div>
    );
}
```

මේ කෝඩ් එකේදී ඔයා බොත්තම ඔබන ඔබන සැරේට Screen එකේ තියෙන අංකය (0, 1, 2, 3...) ඉබේම වෙනස් වෙනවා. ඒක තමයි `useState` වල මැජික් එක! 

අපේ Project එකේ **Form එකක් පුරවද්දී (උදා: ඊමේල් එක සහ පාස්වර්ඩ් එක ටයිප් කරද්දී)** ඒ ටයිප් කරන අකුරු මතක තියාගන්නෙත් මේ `useState` හරහාමයි.

## 2. `useEffect` (තිරයට ආපු ගමන් කරන වැඩේ)

Component එකක් බ්‍රව්සර් එකේ ලෝඩ් වුණ ගමන් (නැත්නම් දිස් වුණ ගමන්) අපිට මොනවා හරි වැඩක් ඉබේම කරගන්න ඕනේ නම්, ඒකට අපි පාවිච්චි කරන්නේ `useEffect` එකයි.

**මේක ගොඩක්ම පාවිච්චි වෙන්නේ කොහෙද?**
අපි කලින් කතා කළා අපේ Frontend එක Backend (PHP) එකෙන් දත්ත ඉල්ලනවා කියලා (උදා: ඉවෙන්ට්ස් ටික ඉල්ලනවා). 
Page එක ලෝඩ් වුණ ගමන්ම Backend එකට කෝල් කරලා (API Request එකක් යවලා) දත්ත අරන් එන්න අපි පාවිච්චි කරන්නේ මේ `useEffect` එකයි.

```tsx
"use client";
import { useState, useEffect } from "react";

export default function EventListPage() {
    // ඉවෙන්ට්ස් ටික මතක තියාගන්න හිස් Array ([]) එකක් හදාගන්නවා
    const [events, setEvents] = useState([]);

    // useEffect එකෙන් කියන්නේ "මේ Component එක ලෝඩ් වුණ ගමන් මේ ටික කරන්න"
    useEffect(() => {
        
        // 1. Backend එකෙන් දත්ත ඉල්ලනවා
        fetch("http://localhost/backend/get_events.php")
            .then(response => response.json()) // ආපු JSON දත්ත ටික කියවනවා
            .then(data => {
                // 2. අරන් ආපු දත්ත ටික අපේ State පෙට්ටියට දාගන්නවා
                setEvents(data);
            });

    }, []); // මේ හිස් කොටු වරහන් වලින් කියන්නේ "මේක එක පාරක් විතරක් කරන්න" කියන එකයි.

    return (
        <div>
            <h1>ඉදිරියට ඇති උත්සව:</h1>
            {/* පෙට්ටියේ තියෙන ඉවෙන්ට්ස් ටික තිරයේ පෙන්වනවා */}
            {events.map(evt => (
                <p key={evt.id}>{evt.title}</p>
            ))}
        </div>
    );
}
```

> **සාරාංශය:**
> * Component එකකට මතකයක් දෙන්න සහ අගයන් වෙනස් වෙද්දී තිරය Update කරන්න **`useState`** පාවිච්චි කරනවා.
> * Component එකක් ලෝඩ් වුණ ගමන්ම ඉබේම මොනවා හරි කරන්න ඕනේ නම් (උදා: PHP එකෙන් දත්ත අරන් එන්න) **`useEffect`** පාවිච්චි කරනවා.

දැන් ඔයාට Frontend එක වැඩ කරන ලොජික් එක තේරෙනවා. ඊළඟ පාඩමෙන් අපි බලමු අපේ UWU-NEXUS එකේ මේ Components අලංකාර කරන්න (ලස්සන කරන්න) අපි පාවිච්චි කරලා තියෙන **Tailwind CSS** තාක්ෂණය ගැන!
