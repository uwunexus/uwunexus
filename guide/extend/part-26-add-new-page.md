# Part 26: ප්‍රායෝගික වැඩ - අලුත් Page එකක් හදමු!

මේ වෙනකන් අපි බැලුවේ දැනට අපේ UWU-NEXUS Project එකේ ලියලා තියෙන කෝඩ් කොහොමද වැඩ කරන්නේ කියලා. හැබැයි ඔයාට දවසක ඕනේ වුණොත් අලුතින්ම Feature එකක් (උදාහරණයක් විදිහට "රුධිර දායකයින්ගේ ලැයිස්තුවක්") වෙබ්සයිට් එකට එකතු කරන්න, ඒක කරන්නේ කොහොමද?

අපි මේ Module 7 එකෙන් ඒක පියවරෙන් පියවර කරලා බලමු. මේ පාඩමෙන් අපි මුලින්ම කරන්නේ **අපේ වෙබ්සයිට් එකට අලුතින් පිටුවක් (Page එකක්) එකතු කරන එකයි.**

---

## 1. ෆෝල්ඩරයක් හදන්න (Routing)

Next.js වල අලුත් පිටුවක් (Route එකක්) හදන එක මහා පුදුම විදිහට ලේසියි. වෙනත් භාෂාවල වගේ ඒකට වෙනම කෝඩ් ලියන්න ඕනේ නැහැ. ෆෝල්ඩරයක් හැදුවාම ඒක ඉබේම Page එකක් බවට පත්වෙනවා!

අපිට අලුත් පිටුවේ ලින්ක් එක **`localhost:3000/blood-donors`** විදිහට හදාගන්න ඕනේ කියලා හිතමු.

**කරන්න තියෙන්නේ මෙච්චරයි:**
1. ඔයාගේ Project එකේ **`app`** ෆෝල්ඩරය ඇතුළට යන්න.
2. ඒක ඇතුළේ අලුතින් ෆෝල්ඩරයක් හදන්න **`blood-donors`** කියලා (අකුරු ඔක්කොම සිම්පල් වලින්, හිස්තැන් වෙනුවට ඉරක් `-` දාලා ලියන්න).

## 2. Page එකේ කෝඩ් එක ලියන්න (`page.tsx`)

අර හැදුව අලුත් ෆෝල්ඩරය (`blood-donors`) ඇතුළට ගිහින්, ඒක ඇතුළේ අලුත් ෆයිල් එකක් හදන්න **`page.tsx`** කියලා. (Next.js වල හැම පිටුවකම නම අනිවාර්යයෙන්ම `page.tsx` වෙන්නම ඕනේ).

දැන් ඒ ෆයිල් එක ඇතුළේ අපේ මූලික React Component එක (ලෙගෝ කෑල්ල) ලියමු:

```tsx
// app/blood-donors/page.tsx

export default function BloodDonorsPage() {
    return (
        <div className="p-8">
            {/* ලොකු මාතෘකාව */}
            <h1 className="text-3xl font-bold text-red-600 mb-4">
                රුධිර දායකයින්ගේ ලැයිස්තුව
            </h1>
            
            {/* පොඩි විස්තරයක් */}
            <p className="text-gray-700">
                හදිසි අවස්ථාවකදී රුධිරය ලබාදිය හැකි සිසුන්ගේ තොරතුරු මෙහි ඇතුළත් වේ.
            </p>

            {/* තවම Backend එක හදපු නැති නිසා නිකන්ම බොරු දත්තයක් පෙන්වමු */}
            <div className="mt-6 bg-white p-4 rounded shadow">
                <h2 className="text-xl font-semibold">නිලේෂ් ජයානන්දන</h2>
                <p>රුධිර ගණය: O+</p>
                <p>දුරකථනය: 077-XXXXXXX</p>
            </div>
        </div>
    );
}
```

## 3. ප්‍රතිඵලය බලමු!

දැන් ඔයා Project එක On කරලා (Command Prompt එකේ `npm run dev` ගහලා) බ්‍රව්සර් එකේ **`http://localhost:3000/blood-donors`** කියන ලින්ක් එකට ගියොත්, ඔයා අර කෝඩ් එකේ ලියපු "රුධිර දායකයින්ගේ ලැයිස්තුව" කියන පිටුව ලස්සනට පේන්න තියෙයි!

> **සාරාංශය:**
> * Next.js වල අලුත් Page එකක් හදන්න ඕනේ නම්, මුලින්ම **`app`** ෆෝල්ඩරය ඇතුළේ ඕනේ කරන නමින් ෆෝල්ඩරයක් හදන්න ඕනේ. (උදා: `app/blood-donors`)
> * ඊටපස්සේ ඒ ෆෝල්ඩරය ඇතුළේ **`page.tsx`** කියලා ෆයිල් එකක් හදන්න ඕනේ.
> * ඒ ෆයිල් එක ඇතුළේ `export default function` එකක් ලියලා ඒකෙන් HTML (JSX) ටික return කරන්න ඕනේ.

දැන් අපි Frontend එකේ ලස්සනට පිටුවක් හැදුවා. හැබැයි මේකේ දත්ත තියෙන්නේ අපේ කෝඩ් එක ඇතුළෙමයි (Hardcoded). මේකට දත්ත එන්න ඕනේ Database එකෙන්. 

ඒ නිසා ඊළඟ පාඩමෙන් අපි බලමු **මේ දත්ත ටික Database එකෙන් අරන් දෙන අලුත් Backend ෆයිල් එකක් (API එකක්) ලියන්නේ කොහොමද** කියලා.
