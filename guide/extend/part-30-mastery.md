# Part 30: අවසානය සහ ඉදිරියට (From Zero to Hero)

සුබ පැතුම්! 🎉 ඔයා දැන් මේ මාර්ගෝපදේශයේ (Guide) අවසානම පාඩමට ඇවිත් ඉන්නේ. 

මීට දවස් කීපයකට කලින් කෝඩ් එකක් දැක්කම "මොනවද මේ අමුතු වචන" කියලා හිතුණු ඔයාට, අද වෙද්දී කෝඩ් එකක් දැක්කම ඒක කියවලා තේරුම් ගන්න පුළුවන් මට්ටමට ඇවිත් තියෙනවා. ඒක ලොකු ජයග්‍රහණයක්!

අපි ආපු ගමන පොඩ්ඩක් ආපහු හැරිලා බලමු.

---

## 1. අපි ඉගෙන ගත්ත දේවල් (The Journey)

**Module 1 & 2: පදනම (The Basics)**
* වෙබ්සයිට් එකක Frontend එක සහ Backend එක කියන්නේ මොකක්ද කියලා අපි ඉගෙන ගත්තා.
* PHP වලින් `echo` කරන හැටි, `if/else` තීරණ ගන්න හැටි සහ `foreach` ලූප් දාන හැටි ඉගෙන ගත්තා.
* MySQL Database එකක් කියන්නේ Excel Sheet එකකට සමාන දෙයක් කියලත්, ඒකෙන් දත්ත අරන් එන්න (`SELECT`) සහ දාන්න (`INSERT`) අපි ඉගෙන ගත්තා.

**Module 3 & 4: Database සහ ගෘහ නිර්මාණය (Architecture)**
* OOP කියන මහා සංකල්පය කාර් එකක් හදන ෆැක්ටරියකට සමාන කරලා අපි ඉගෙන ගත්තා. (Class සහ Object).
* අපේ Project එකේ පෙනුම වෙනස් කරන්නේ `app/` එකෙන් කියලත්, ලොජික් වෙනස් කරන්නේ `backend/` එකෙන් කියලත් අපි තේරුම් ගත්තා.
* Primary Key සහ Foreign Key හරහා Tables එකිනෙකට සම්බන්ධ වෙන හැටි ඉගෙන ගත්තා.

**Module 5 & 6: ඇත්තම ලෝකයට! (Real World Features)**
* React Components (ලෙගෝ කෑලි), `useState` (මතකය) සහ `useEffect` (ඉබේ වැඩ කිරීම) ගැන ඉගෙන ගත්තා.
* ඊටපස්සේ අපි Signup, Login, Marketplace, Events වගේ Features අපේ Frontend එක සහ Backend එක එකතු වෙලා වැඩ කරන විදිහ පේළියෙන් පේළිය කියවලා බැලුවා.

**Module 7 & 8: ඔයාගේම වැඩක්! (Extending & Debugging)**
* අලුතින්ම Database Table එකක් හදලා, ඒකට අලුත් PHP API එකක් ලියලා, අලුත් Next.js Page එකකට ඒ දත්ත ගෙන්නගන්න හැටි ඔයා තනියම ඉගෙන ගත්තා.
* ප්‍රශ්නයක් ආවම බ්‍රව්සර් එකේ F12 (Developer Tools) ගහලා ඒක විසඳගන්න (Debug) කරන හැටිත් ඉගෙන ගත්තා.

## 2. ඊළඟට මොකද කරන්නේ? (Next Steps)

කෝඩින් (Coding) කියන්නේ පොත් කියවලා හරි, මේ වගේ Guides කියවලා විතරක් හරි ඉගෙන ගන්න පුළුවන් දෙයක් නෙවෙයි. මේක හරියට බයිසිකල් පදින්න ඉගෙන ගන්නවා වගේ වැඩක්. වැටි වැටී පදින්නම ඕනේ.

1. **කෝඩ් එක වෙනස් කරලා බලන්න:** 
   අපේ `app/login/page.tsx` එකට ගිහින් බොත්තමේ තියෙන "Login" කියන වචනේ "ඇතුළු වන්න" කියලා වෙනස් කරලා බලන්න. ඒකෙන් මුකුත් කැඩෙන්නේ නෑ.
2. **අලුත් දේවල් හදන්න:**
   අපි Part 26, 27, 28 වලදී රුධිර දායකයින්ගේ ලැයිස්තුවක් හැදුවා වගේ, ඔයා තනියම "කැම්පස් එකේ කැන්ටිමේ කෑම ලිස්ට් එක" පෙන්වන Page එකක් හදලා බලන්න උත්සාහ කරන්න.
3. **AI එකෙන් උදව් ගන්න:**
   මොනවා හරි තේරෙන්නේ නැති තැනක් ආවොත් හරි, කෝඩ් එකක් වැඩ කරන්නේ නැතුව Error එකක් ආවොත් හරි, බය නැතුව මගෙන් (ChatGPT / Gemini වගේ AI කෙනෙක්ගෙන්) අහන්න.
